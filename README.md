

# StudyHive

StudyHive  is just a studying platform that helps to study online in a focused and efficient manner.


## Note

This project shows implementation of PERN stack , socket.io, webrtc to create a online study platform which
offers focused study session while making sure all the necessary features and tools like timer, todo list, notes, chats, ai-assistant, etc

## Deployment

To deploy this project run

```bash
# server
  npm i
  npm run start

# client
    npm i
    npm run dev
```


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file in server side


`DATABASE_URL = 'USED A POSTGRES DB LOCALLY !'`
`CLOUDINARY_CLOUD_NAME = "abc"`
`CLOUDINARY_API_KEY = ''`
`CLOUDINARY_API_SECRET = ''`
`GROQ_API_KEY = ''`
`GoAuth_clientID = ''`
`GoAuth_clientSecret = ''`
